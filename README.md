💰 VoiceBudget – Smart Expense Tracker App

VoiceBudget is a modern and intelligent expense tracking Android application designed to help users manage their finances efficiently using manual input and voice-based expense logging. The app provides real-time insights into spending habits using charts, limits, and AI-based suggestions.

🚀 Features
💸 Expense Management
➕ Add Expense manually
🎤 Add Expense using Voice Input
👀 View all expenses
📜 View detailed expense history
📊 Analytics & Insights
📈 Pie Chart visualization of expenses
📉 Donut Chart for category-wise spending
📊 Trading-style expense trends (spending analysis over time)
💡 Budget Control
🎯 Set monthly/weekly spending limit
📊 View remaining budget limit
⚠️ Alerts when limit is exceeded
🧠 Smart Features
🎤 Voice-based expense entry
🤖 AI-based spending insights & suggestions
🔍 Smart search & scan option for quick entry
🛡️ Additional Features
🏷️ Category tagging system
🔐 Secure and user-friendly design
⚡ Fast and lightweight performance
🧾 Trademark / branding section for app identity
🛠️ Tech Stack
Kotlin / Java (Android)
Jetpack Compose / XML UI
SQLite / Room Database
MPAndroidChart (for Pie & Donut charts)
Speech-to-Text API (Voice feature)
AI integration (for insights)
📱 App Highlights
Smart voice expense tracking
Real-time budget monitoring
Beautiful UI with modern charts
AI-powered financial suggestions
Easy and fast expense logging
🎯 Purpose

The goal of VoiceBudget is to simplify personal finance tracking by combining traditional expense management with modern AI and voice technologies, making budgeting effortless and intelligent.
