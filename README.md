## 📖 Overview

**VoiceBudget** is a modern and intelligent Android expense tracking application that helps users manage their finances efficiently through both manual expense entry and voice-based expense logging.

The app provides real-time insights into spending habits using interactive charts, budget monitoring, and AI-powered financial suggestions.

---

## 🚀 Features

### 💸 Expense Management

* ➕ Add expenses manually
* 🎤 Add expenses using voice input
* 👀 View all expenses
* 📜 Detailed expense history tracking

### 📊 Analytics & Insights

* 📈 Pie chart visualization of expenses
* 🍩 Donut chart for category-wise spending
* 📉 Trading-style expense trend analysis
* 📊 Real-time spending insights

### 💡 Budget Control

* 🎯 Set monthly spending limits
* 🎯 Set weekly spending limits
* 📊 Track remaining budget
* ⚠️ Instant alerts when limits are exceeded

### 🧠 Smart Features

* 🎤 Voice-based expense logging
* 🤖 AI-powered spending suggestions
* 🔍 Smart search for quick expense entry
* 📷 Scan option for faster data input

### 🛡️ Additional Features

* 🏷️ Category tagging system
* 🔐 Secure and user-friendly interface
* ⚡ Fast and lightweight performance
* 🎨 Professional branding and identity section
